## How to Run

1. **Clone the repo**  
```bash
git clone https://github.com/yourusername/Recognizing-handwritten-digits.git
cd Recognizing-handwritten-digits
```

2. **Install dependencies**  
```bash
pip install -r requirements.txt
```

3. **Run the app**  
```bash
python app.py
```
